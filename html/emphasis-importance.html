2. Strong Importance (<strong>)

Use when text carries semantic weight, not just for bold styling.

3. Nesting <strong> & <em>

This is very important—please remember!

4. Deprecated Presentational Tags

❌ Avoid: <big>Big text</big> or <font>color="red">Red</font>

✅ Use CSS instead for presentation.

5. Repurposed Tags (<i>, <b>, <u>)

The Panthera leo is known as the lion.

He shouted, ¡Hasta la vista!

Learn to recieve the correct way.

Underlines imply links—use <u> sparingly or consider CSS.

6. When to Use CSS Instead

Important: this is styled via CSS, not HTML tags.
